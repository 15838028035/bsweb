 5. Good first issue - 容易解决
 6. Needs splitting - 待拆分
 7. Question - 问题
 8. Technical - 技术问题
 9. Under construction - 工作中
 10. Under review - 待评审
 11. Wont fix - 不解决
 12. Needs discussion - 待讨论
